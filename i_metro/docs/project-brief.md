# I-Metro Project Brief (Working Notes)

## Goals
- Build user mobile app (iOS + Android) based on provided UI screens.
- Build admin dashboard (web) to control user app activities.
- Integrate QR validation with validator machines and MoniePoint POS QR.

## Current Decisions
- Preferred stack: Flutter for mobile; Flutter Web for admin (single Dart stack).
- IDE: VSCode.

## Open Items
- Validator SDK details and integration flow.
- Backend/API design (auth, tickets, payments, routes, users, admin roles).
- Exact screen order and navigation flow.
- Branding: fonts, colors, icon set, and final app name/package id.
